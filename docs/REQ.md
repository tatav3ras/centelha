# REQ funcionais

- Sistema deve Autenticar o responsável familiar
- Sistema deve ter Cadastro de famílias e beneficiários
- Sistema deve ter Cadastros dos benefícios disponíveis gerenciavel pelo sistema
- Sistema deve ter Controle de benefícios por família
- Sistema deve ter Historico de beneficios por data e familia
- Sistema deve ter Dashboard de vulnerabilidade por região
- Sistema deve ter Alertas de vencimento de benefícios e que não atendidos
- Sistema deve ter Relatórios de benefícios por região e tipo
- Sistema deve ter panel administrativo

Telas:

- Tela de Login
- Tela de Cadastro de Famílias
- Tela de Cadastro de Beneficiários
- Tela de Controle de Benefícios
- Tela de Histórico de Benefícios
- Tela de Dashboard de Vulnerabilidade
- Tela de Alertas
- Tela de Relatórios
- Tela Administrativa
